# PyCharm Quickstart

## Installation

1. Go to the [PyCharm download page](https://www.jetbrains.com/pycharm/download), make sure to scroll down a bit so you don't accidentally download the Professional Edition.
1. Select "Apple Silicon" in the Download dropdown
1. Open the `.dmg` file and drag the PyCharm CE icon into your Applications folder
1. Make sure to eject the PyCharm CE disk image by clicking the triangle icon to the right of it
1. Use Spotlight Search to start up PyCharm CE
1. Miscellanous
   1. Confirm that you really want to open PyCharm
   1. Agree to the terms
   1. Choose "Don't Send" for data sharing
   1. Skip import
